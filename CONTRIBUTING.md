# Contributing to Weather Dashboard

Thank you for your interest in contributing! Here are some guidelines to help you get started.

## How to Contribute

1. Fork the repository and create your branch from `main`.
2. If you've added code, please test it thoroughly.
3. Follow the existing code style (we use Prettier and ESLint – run `npm run lint` if configured).
4. Write clear, descriptive commit messages.
5. Open a pull request with a detailed description of your changes.

## Reporting Bugs

- Use the GitHub Issues tab.
- Include steps to reproduce, expected behavior, and screenshots if possible.
- Mention your environment (browser, OS).

## Suggesting Features

- Open an issue with the label `enhancement`.
- Explain why the feature would be useful and how it might work.

## Development Setup

Refer to the README for installation instructions.  
Start the dev server with `npm run dev`.

## Code of Conduct

Please read our [Code of Conduct](CODE_OF_CONDUCT.md). We expect all contributors to follow it.

Thank you for making this project better!
