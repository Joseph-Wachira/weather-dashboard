# Weather Dashboard

A beautiful, responsive weather dashboard built with React, Vite, Tailwind CSS, and OpenWeatherMap API. Features dynamic backgrounds, 5-day forecast, search history, and smooth animations.

## Features

- Search any city worldwide
- Current weather (temperature, feels like, humidity, wind, pressure)
- Sunrise and sunset times
- 5-day forecast with min/max temperatures
- Recent search history (stored in localStorage)
- Loading spinner and error handling
- Dynamic backgrounds based on weather (clear, clouds, rain, snow, thunderstorm, mist, night)
- Fully responsive design
- Glassmorphism UI with framer-motion animations
